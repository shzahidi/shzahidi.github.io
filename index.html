My Background

I graduated from Indiana University in December 2019 with a Bachelor of Science degree in Informatics and a specialization in Health Informatics from the Luddy School of Informatics, Computing, and Engineering. I have participated in three National Science Foundation Research Experience for Undergraduates Programs (NSF - REU) during the summer of each of my years of undergraduate study as a Student Researcher. I have also worked for Technology Center Consulting at Indiana University as a Technical Consultant and a Consultant Supervisor. I am currently working for the IU Computer Vision Lab as a Research Assistant on a project analyzing the health and behavior of animals in uninterrupted and interrupted biomes.

My Projects

Cloudmesh.Robot

This project consists of the implementation of a Marvelmind Navigation System for autonomous control of NodeMCU robot cars on ESP8266 microchips, Arduino microprocessors and Raspberry PI 3. This project is part of the National Science Foundation's Cloudmesh partnership with the Smith Research Center.

  • GitHub Repository

Weather Application

This is a weather emergency response application that I worked on for presentation at the 2019 Informatics in Disaster and Emergency Response (IDER) symposium at Indiana University. It showcases weather radar and traffic feedback programmed in HTML and JavaScript and is fully compatible with EOC displays and Raspberry PI 3.

IoT Course Coordination

This project contains a documentation set that I worked on for the course INFO-I 523: Big Data Applications and Analytics that also acts as a book chapter for Ph.D candidates participating in the course for credit towards a Ph.D in Computer Science from the Indiana University - Luddy School of Informatics, Computing, and Engineering. The documentation set consists of instructions on installing, maintaining, and implementing common IoT packages as well as python modules I worked on for Arduino kits.

  • Documentation Set

Cyrptographic Access Scheme Implementation

This is a National Science Foundation funded privacy engineering project that I participated in, working to secure an Indiana University - Health staff database during the Summer of 2019. The project utilizes a Django facial-recognition REST API to allocate role-based privilege allocation to manage the credentials and access of hospital staff, as well as utilizing the Django Web Framework for server-side encryption and privilege designation.
